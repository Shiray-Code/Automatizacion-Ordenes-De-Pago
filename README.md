# Automatización Órdenes de Pago

Automatización desarrollada en Python para procesamiento de órdenes de pago, extracción de datos desde PDFs y generación de reportes operacionales.

---

## Objetivo

Reducir trabajo manual en el procesamiento de documentos financieros mediante automatización de extracción, validación y consolidación de información.

---

## Funcionalidades

- Lectura automática de PDFs
- Extracción de información de facturas y boletas
- Consolidación de datos
- Automatización de generación de reportes
- Validación de montos y datos operacionales
- Integración con Excel

---

## Tecnologías utilizadas

- Python
- Json
- OpenPyXL
- PDFPlumber
- Playwright
- Excel

---

## Resultado

La automatización permitió disminuir significativamente el tiempo dedicado al procesamiento manual de órdenes de pago y reducir errores operacionales.
